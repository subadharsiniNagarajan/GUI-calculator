# GUI Calculator (Python)

This is a simple GUI calculator created using Python and Tkinter.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Easy to use interface

## Technologies Used
- Python
- Tkinter (GUI Library)

## How to Run the Project

1. Make sure Python is installed in your system.
2. Download or clone this repository.
3. Open terminal / command prompt.
4. Go to the project folder.
5. Run the file using:

   python calculator.py

## Example Usage
- Enter first number
- Select operation
- Enter second number
- Click "=" to get result
